# Simple Todo List App

A clean, responsive, and intuitive Todo List application built using pure HTML, CSS, and modern JavaScript. This project is designed to help users manage their daily tasks efficiently with a minimalist and user-friendly interface.

## 🚀 Live Demo

[Check out the live application here!](https://melodious-monstera-82b6d1.netlify.app/)

---

## ✨ Features

* **Data Persistence:** Tasks are securely saved so they never vanish when you refresh the page.
* **Add Tasks:** Quickly add new items to your todo list.
* **Complete Tasks:** Mark tasks as completed with a visual strike-through.
* **Delete Tasks:** Remove individual tasks when they are no longer needed.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop viewports.

---

## 🛠️ Tech Stack

* **HTML5:** Semantic markup for a structured webpage layout.
* **CSS3:** Custom styling and responsive design without external frameworks.
* **JavaScript (ES6+):** Dynamic DOM manipulation, state management, and persistent storage logic.
